nom : ajouter un intérêt
but : un professeur veut ajouter un intérêt
acteur : professeur (principal)
précondition: avoir un compte professeur

Scénario principal:

	1. le professeur se connecte
	2. le professeur clique sur l'option "Ajouter un intérêt"
	3. le professeur se fait demander d'ajouter le titre de l'intérêt
	4. le professeur se fait demander d'ajouter un description de l'intérêt
	5. l'intérêt complétée est montrée au professeur par le système
	6. le professeur confirme la création
	7. le professeur revient sur la page d'acceuil

Scénario alternatif:

	 
	6.1.1 le professeur clique sur l'option "Changer le titre"
	6.1.2 le professeur clique sur l'option "Changer la description"
	6.2 retour à l'étape 5
	
	6.1.3 le professeur annule la création
