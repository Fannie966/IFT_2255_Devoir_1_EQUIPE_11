nom : suspendre un membre
but : un administrateur veut suspendre le compte d'un membre
acteur : administrateur (principal)
précondition : avoir un compte administrateur

Scénario principal:

	1. l'administrateur se connecte
	2. l'administrateur choisi l'option "Changer l'état d'un compte"
	3. l'administrateur entre le nom du membre qu'il veut suspendre
	4. le système vérifie que le membre existe dans la BD
	5. le système vérifie que le membre n'est pas suspendu
	6. le système montre un message disant : "Voulez-vous suspendre le comtpe de XXXXXXX?"
	7. l'administrateur confirme le choix
	8. l'administrateur revient sur la page d'acceuil

Scénario altérnatif:

	4.1 le membre n'existe pas et le système montre un message d'erreur à l'administrateur
	4.2 retour à l'étape 3	
	5.1 le système se rend compte que le compte est déjà suspendu
	6.1 le système montre un message disant : "Voulez-vous activer le compte de XXXXXXX?"
	7.1 l'administrateur annule la requête
